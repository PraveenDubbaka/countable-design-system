---
name: Navy Core Theme
description: Navy Core is the permanent and only theme; the ThemeChanger UI and presets are removed
type: constraint
---
The application uses the Navy Core theme exclusively, defined in `src/index.css`. The floating ThemeChanger panel and all preset themes (Midnight, Deep Gradient, etc.) have been removed. Do not re-add a theme switcher or alternative presets. **Why:** product decision to lock branding to Navy Core.
